# GMUP

Site du Groupe Marocain des Urgences Pédiatriques.

## Stack

- Next.js 16
- React 19
- Tailwind CSS v4
- Framer Motion + Lenis

## Pages

| Route | Contenu |
|-------|---------|
| `/` | Accueil — Mot de la Présidente + Bureau |
| `/blog` | Espace public / blog |
| `/evenements/[slug]` | Détail d'un événement |

## Démarrer

```bash
npm install
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000).
